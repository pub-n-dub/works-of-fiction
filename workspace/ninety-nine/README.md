# Ninety-Nine Workspace

This workspace is the clean production area for preparing *Ninety-Nine*
for editorial completion and eventual professional publication.

## Goal

Move from archive materials to one publishable edition with:

- one canonical manuscript
- a clear record of incomplete or missing sections
- one edited working manuscript
- cleaned metadata and front matter
- production-ready ebook and print files once the text is complete

## Status

`Ninety-Nine` is not yet publication-ready.

The current manuscript is substantial and coherent enough to begin
editorial work, but it is known to be incomplete. Per current project
understanding, the missing major gap is the final `thirty days` section.

## Working Rule

Treat the existing `Ninety-Nine/` directory as source/archive material.
Treat this workspace as the place where current editorial and production
decisions live.
