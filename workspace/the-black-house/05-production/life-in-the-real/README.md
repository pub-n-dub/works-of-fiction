# Life in the Real Output

This directory is the standalone output area for `Life in the Real or the Transformation of a "Normal" Person Through the Void of an Enigmatic Existence`.

## Files

- `life-in-the-real-original.md`: archival extract from the working manuscript
- `life-in-the-real-edited.md`: editorial working file
- `life-in-the-real.md`: current build source
- `build-life-in-the-real-pdf.sh`: build helper for a standalone PDF

## Purpose

Use this output when revising, assessing, or typesetting this story apart from the rest of *The Black House*.

`life-in-the-real-original.md` preserves the extracted source as-is.

`life-in-the-real-edited.md` preserves the current editorial working version.

`life-in-the-real.md` is the file used to build the standalone review PDF.

## Build

From this directory:

```sh
./build-life-in-the-real-pdf.sh
```

This creates:

- `life-in-the-real.tex`
- `life-in-the-real.pdf`
