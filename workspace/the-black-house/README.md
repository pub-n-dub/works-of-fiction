# The Black House Workspace

This workspace is the clean production area for preparing *The Black House* for professional publication.

## Goal

Move from archive materials to one publishable edition with:

- one canonical manuscript
- one agreed table of contents
- cleaned metadata and copyright information
- production-ready ebook and print files
- cover, copy, and retailer-facing materials

## Working Rule

Treat the existing `The Black House/` directory as source/archive material.
Treat this workspace as the place where current publishing decisions live.

## Folder Guide

- `00-intake/`: source inventory, notes on what exists, and what we are treating as canonical
- `01-manuscript/`: the live manuscript and structural notes
- `02-editorial/`: edit letters, line-edit notes, proofing, open issues
- `03-metadata/`: title page, copyright page, BISAC, keywords, author/imprint details
- `04-design/`: cover and interior design assets
- `05-production/`: export-ready ebook and print packages
- `06-marketing/`: description, author bio, back-cover copy, launch notes
- `07-rights/`: ISBNs, rights notes, permissions, publishing platform records
- `08-archive/`: retired workspace drafts and superseded materials

## Immediate Next Steps

1. Lock the table of contents and included stories.
2. Clean author/publisher metadata.
3. Define front matter and back matter.
4. Create the working edited manuscript from the canonical source.
5. Build polished ebook and print editions from the cleaned manuscript.
