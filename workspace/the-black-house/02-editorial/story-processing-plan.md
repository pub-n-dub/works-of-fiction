# Story Processing Plan

This plan defines the repeatable workflow for moving the remaining stories in *The Black House* toward a publication-ready edited book.

## Goal

Process each remaining story the same way `Prelude` was processed so that, at the end, the collection has:

- a clean standalone output for each story
- an archival extract and an editorial working file for each story
- notes documenting any AI-assisted formatting or copy changes
- a standalone PDF path for review
- enough editorial groundwork to support a final book-level revision pass

## Standard Per-Story Workflow

For each story:

1. Extract the story from the canonical working manuscript.
2. Create a standalone output folder.
3. Preserve an archival `*-original.md` version.
4. Create an editorial `*-edited.md` version.
5. Create a build source and standalone PDF path.
6. Add AI-change notes if formatting or punctuation changes are introduced.
7. Assess the story editorially against the line-edit checklist.
8. Preserve the distinction between archival source and working edited text.

## Processing Order

1. `A Forgotten Memory`
2. `Absolution`
3. `Testimony of a Loved One`
4. `Of Something I Once Knew`
5. `Life in the Real or the Transformation of a "Normal" Person Through the Void of an Enigmatic Existence`
6. `The Electric Cipher`
7. `The Flashback`
8. `The Itch`

## Completion Criteria

The book reaches story-processing completion when all eight remaining pieces have:

- standalone output folders
- original and edited manuscript files
- buildable standalone review PDFs
- AI-change notes where applicable

After that, the collection still needs:

- story-level revision passes
- collection-level sequencing and framing decisions
- front matter and metadata cleanup
- a final integrated production pass
